# Velocity 模板引擎

Apache Velocity 是一款基于 Java 的后端模板引擎，诞生于 Apache 软件基金会。它提供了一套简单而强大的模板语言（VTL，Velocity Template Language），核心设计理念是强制分离业务逻辑与视图表现，允许将 Java 对象中的数据无缝渲染成 HTML、XML、SQL、JSON 或纯文本等任意文本格式。

虽然在前端渲染领域它已非主流，但在后端代码生成、自动化脚本生成、邮件模板等场景中，它依然是 Java 生态中的经典利器。

# 将 Date 类型改为 LocalDateTime

Date 类型与 LocalDateTime 的区别？

  1. 不可变性与线程安全性
    1. Date 一旦创建 Date 对象，可以通过 setTime (long time) 进行修改，多线程下存在线程安全问题
    2. LocalDateTime 是不可变的，线程安全
  2. 时间精度
    1. Date 可以表示到毫秒的时间值
    2. LocalDateTime 可以表示到纳秒的时间值
  3. 易用性：LocalDateTime 的 API 设计的更现代化，易于使用
