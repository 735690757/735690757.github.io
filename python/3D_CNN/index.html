# 图像的本质

图像是由 RGB 三通道组成的，一般数值范围是 0 到 255.

因此像素矩阵为：H * W * 3，其中 H 为高，W 是宽。

# 从图像到视频

视频本质上是由连续的图片（视频帧），快速播放构成的。

图片的像素矩阵是 H * W * 3

视频就是 D * H * W * 3，其中 D 是深度，是视频帧的叠加。

不同的库对与这 4 个维度顺序表述不同，但这四个维度诚然如此。
