# 数据生成

第一次探索先生成 2000 训练样本，其中训练比例为 8.5：1.5

gz=Gρ{−(x−ξ)ln⁡[r+(y−η)]−(y−η)ln⁡[r+(x−ξ)]+(z−ζ)arctan⁡(x−ξ)(y−η)(z−ζ)r}∣ξ1ξ2∣η1η2∣ζ1ζ2g_z = G\rho \left\{ -(x-\xi) \ln[r+(y-\eta)] -(y-\eta) \ln[r+(x-\xi)] + (z-\zeta) \arctan \frac{(x-\xi)(y-\eta)}{(z-\zeta)r} \right\} \bigg|_{\xi_1}^{\xi_2} \bigg|_{\eta_1}^{\eta_2} \bigg|_{\zeta_1}^{\zeta_2} gz​=Gρ{−(x−ξ)ln[r+(y−η)]−(y−η)ln[r+(x−ξ)]+(z−ζ)arctan(z−ζ)r(x−ξ)(y−η)​}∣∣∣∣∣​ξ1​ξ2​​∣∣∣∣∣​η1​η2​​∣∣∣∣∣​ζ1​ζ2​​

gzz=Gρarctan⁡(x−ξ)(y−η)(z−ζ)r∣ξ1ξ2∣η1η2∣ζ1ζ2g_{zz} = G\rho \arctan \frac{(x-\xi)(y-\eta)}{(z-\zeta)r} \bigg|_{\xi_1}^{\xi_2} \bigg|_{\eta_1}^{\eta_2} \bigg|_{\zeta_1}^{\zeta_2} gzz​=Gρarctan(z−ζ)r(x−ξ)(y−η)​∣∣∣∣∣​ξ1​ξ2​​∣∣∣∣∣​η1​η2​​∣∣∣∣∣​ζ1​ζ2​​

这两个是地球物理重力正演中非常经典的 ** 矩形棱柱体重力异常（gzg_zgz​）及其导数（gzzg_{zz}gzz​）** 的解析表达式。gzg_zgz​ 作为通道 1，gzzg_zzgz​z 作为通道 2 输入到网络之中

# 并不严谨的结论

当前，我还没有应用一些评价指标，以及控制对比变量，只是直观的探讨了一下模型。就这六次来讲，UNet_ResNeXt50_32x4d (4) 与 Att_Unet with GLoss (6) 相对来讲是最好的，后续将会把 ResNeXt50 主干融入到 UNetpp 与 Att_UNetpp 之中，以及近期的新的主干 ViT 与 ViM，将会提上日程。除此之外，选择或创造更适用于反演任务的模块也尤为重要。
