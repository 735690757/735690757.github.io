# NumPy 核心处理方法

NumPy（Numerical Python）是一个用于科学计算的 Python 第三方库，它提供了：

# 核心功能：

  1. 多维数组对象
    • 高效地存储和操作大规模数字数据。
    • 比原生 Python 的列表（list）更快、更节省内存。
  2. 广播功能
    • 允许不同形状的数组进行数学运算，简洁高效。
  3. 数学函数库
    • 包括线性代数、傅里叶变换、随机数生成、统计分析等。
  4. 数组索引与切片
    • 支持布尔索引、花式索引，比 Python 原生更灵活强大。
